# NOFA Founder Hub

Light-theme founder landing page for Farhad Nasserghodsi and NOFA AI Factory.

## Included
- `index.html` — static landing page
- `assets/farhad.png` — portrait image
- `assets/nofa-logo.png` — logo
- `assets/nofa-factory-scene.png` — NOFA AI Factory visual
- `assets/careerpilot-qr.png` — QR code for CareerPilot AI

## Deploy
1. Create a GitHub repo.
2. Upload all files in this folder.
3. Connect the repo to Vercel.
4. Set the project to deploy from the repo root.

## Update later
- Change the highlighted SaaS link in `index.html`
- Update the countdown launch date near the bottom of the file
- Swap images in `/assets` without changing file names
